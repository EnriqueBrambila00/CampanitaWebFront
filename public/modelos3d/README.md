# Modelos 3D para La Campanita (.glb / .gltf)

Coloca aquí tus archivos de modelos 3D (exportados desde Blender, Maya, o Unity en formato **.glb**).

## ¿Cómo usarlos en el juego?
1. Copia tu archivo aquí, por ejemplo: `guerrero_mictlan.glb`.
2. Ve al **Dashboard Admin** de la página, a la sección **Personajes**.
3. En el campo **"URL Imagen / Modelo 3D (.glb)"**, escribe la ruta local:
   ```
   /modelos3d/guerrero_mictlan.glb
   ```
4. ¡Guarda el registro! Automáticamente la pestaña de Personajes y el Dashboard mostrarán tu modelo 3D interactivo en rotación 360° con controles de cámara.
